- [Burlington, VT](https://philschatz.com/forecast/#44.492585,-73.2617536)

- [Portland, OR](https://philschatz.com/forecast/#45.5270807,-122.7181832)

To check another location change the lat/long coordinates in the URL. You can obtain them from websites like [openstreetmap](https://www.openstreetmap.org), wikipedia, or other commercial map sites.
